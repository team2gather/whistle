# Koa Knex React Typescript Heroku Boilerplate

This is the boilerplate codebase for deploying a back-end react-react typescript on heroku

## Get Started

Development
```zsh
yarn
npm watch-server
```

### Manually Building
```zsh
yarn
npm run build-ts
npm run start
```
